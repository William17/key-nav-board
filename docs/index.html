key-nav-board

一个使用键盘对页面上的块进行导航（焦点控制）的库，可与 React，Vue 和 Svelte 等集成。尤其适用于大屏端页面的焦点控制。

  • 0 依赖，体积小，大约 3kb
  • 低层，方便与不同的库集成和在不同环境使用
  • 可与现代前端框架集成方案
